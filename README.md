# Encore
Encore is a Procedural Music Generation roguelike that blends real-time rhythm gameplay with strategic card choices.

## How to Play

Download the binary for your operating system from the `game/` folder and run it.

## Setup (for developers)

1. Clone the repo: `git clone https://github.com/luizz-couto/encore.git`
2. Install [Godot 4.6](https://godotengine.org/download/)
3. Open the `encore/` folder as a project in Godot
4. Hit Play

## Core Gameplay
Procedural Music Generation game that combines real time rhythm gameplay with rogue like elements. The music plays infinitely within a subset of different electronic music genres, each with its own personality and instruments. The player must hit each instrument’s notes with precision, while the music evolves based on time and the player’s choices.
