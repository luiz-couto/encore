# Encore
Encore is a Procedural Music Generation roguelike that blends real-time rhythm gameplay with strategic card choices.

## Core Gameplay
The game's core mechanic is rhythm-based. The player interact with a 3D board displaying a series of notes.

<img width="602" height="374" alt="image" src="https://github.com/user-attachments/assets/7698a50c-e9fb-4ac9-98d8-e681d533a78e" />

Each song will have a minimum score requirement, which varies based on the song's difficulty and the player's progress in the run. If the player doesn't achieve the required score, they get one more chance, called an "Encore", to try again during that run. If the Encore has already been used, the run ends, the player collects their rewards, and starts over.

<img width="486" height="200" alt="image" src="https://github.com/user-attachments/assets/f23853c1-1524-494e-9ef4-e3404e27b9a2" />

Since the music in Procedural Generated, the player will have the option to alter the song characteristics during its gameplay.

## The Map

At the start of the run and after each song, the player will be presented with the Map, in which they can choose different paths to take. Different paths means different types of songs.

<img width="651" height="370" alt="image" src="https://github.com/user-attachments/assets/71191491-93d9-4d81-98a6-c2dbd7e8b109" />

In the shop you can buy permanent modifications. Which means taht the items bought here can be used in following runs, to adapt the gameplay and to give the player more chances to win.
